# standardbankchallenge
Standard Bank Challenge

Technologies used 

- Spring boot
- H2 (data storage)
- liquibase (migration tools)
- github actions (ci/cd)
- Restfull api
- github authprovider
- docker
- aws EC2
- swagger

Functionalities

- fetch list of books , to get ISBNs
- add order (with ISBN fetched)
- retrieve book Orders (with paginations)
